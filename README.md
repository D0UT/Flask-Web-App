# Notes Web App

## Live Demo

Go to https://D0UT.pythonanywhere.com


## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```
## Viewing The App

Go to `http://127.0.0.1:5000`
